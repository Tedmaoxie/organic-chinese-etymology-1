# 🎓 项目名称
《有机官能团 · 说文解字可视化教学系统》
Organic Functional Groups × Chinese Etymology Visualization Lab

---

## 🧠 项目定位

构建一个面向高中化学教学与教研展示的沉浸式交互网页系统。系统融合：

- **有机官能团结构**：3D球棍模型，支持旋转缩放，官能团关键部位高亮显示。
- **中文命名历史演进**：精准还原 1908–1932 年间从虞和钦的“意译方案”到郑贞文“标准方案”的演进历程（基于何涓论文考据）。
- **汉字构字法解析**：基于学术文献的构字逻辑拆解（形声/会意/象形），拒绝杜撰。
- **偏旁拆解与字源可视化**：点击偏旁触发其从甲骨文到楷书的演化动画，并关联对应的 3D 分子特征。
- **毛笔笔画书写动画**：带墨迹扩散效果的汉字书写演示，配合标准普通话语音发音。

系统需达到**教研展示级视觉质量**，兼具学术严谨性与科技沉浸感，适用于公开课、科普展示等场景。

## 🎨 一、整体视觉风格（展厅级标准）

### 风格关键词
- **玻璃拟态 2.0（Glassmorphism 2.0）**：半透明背景、边缘微光（Rim Light）、拟真折射效果。
- **深色理性美学**：背景采用深空蓝 (#050a15) 到深海蓝 (#0a1f3d) 的极光渐变（Aurora Gradient）。
- **动态化学场粒子系统**：粒子模拟原子微粒，连线模拟化学键感，随鼠标交互产生扰动与汇聚。
- **未来感 / 理性美学**：几何线条、微动效、冷光描边（#00aaff, #0066cc）。

## 🧪 二、核心模块设计与学术考据

### 1️⃣ 3D 分子与汉字联动交互
- **联动触发**：当用户点击汉字偏旁（如“火”或“酉”）时，3D 模型中对应的原子或官能团产生脉冲波纹高亮。
- **视觉叠影**：在展示古字形演化时，背景半透明浮现对应的历史文物图像（如展示“酉”部时浮现商代青铜酒尊）。

### 2️⃣ 汉字构形考据（严格遵循何涓论文）
- **烷 (wán)**：火 + 完。“完”意为“完足、饱足”，指碳原子的四个价键完全被氢原子饱和（饱和烃）。
- **烯 (xī)**：火 + 稀。“稀”意为“稀少”，指氢原子数较烷烃稀少（不饱和烃）。
- **炔 (quē)**：火 + 夬。“夬”取“缺”意，指氢原子数比烯烃更进一步缺失。
- **醇 (chún)**：酉 + 享。本义为“厚酒”，化学中取其“酒之精粹”意。
- **醛 (quán)**：酉 + 旦/全。源自“酒味变也”，暗示醇氧化后的性质转变。
- **酮 (tóng)**：酉 + 同。既是 ketone 的音译，又取古字“酒欲酢（变酸）”之意。
- **醚 (mí)**：酉 + 迷。因其早期作为麻醉剂（使人昏迷）的特性而命名。
- **酯 (zhǐ)**：酉 + 脂。因其广泛存在于油脂（脂肪）中而得名。

## 🧩 三、页面结构与交互流程

### 1️⃣ 首页封面
- 标题：《有机官能团的说文解字》
- 副标题：从分子结构到汉字构形的跨学科可视化研究。
- 视觉：全屏动态极光背景 + 粒子系统，进入按钮带毛玻璃发光效果。

### 2️⃣ 官能团导航区
- 交互卡片水平滚动，展示：中文名、英文名、分子式。
- 点击卡片平滑滚动至对应的详情区，背景极光随官能团特性产生色相偏移（如醇-翠绿、酮-深紫）。

### 3️⃣ 官能团详情区（左右分栏布局）
- **左侧**：Three.js 3D 球棍模型，支持旋转缩放，原子带菲涅尔反射材质。
- **右侧**：
  - **命名来源时间轴**：1908 虞和钦（描述性译名）→ 1920 科学名词审查会 → 1932 郑贞文（标准译名）。
  - **汉字构形拆解**：浮起动画展示偏旁，展示学术考据的构字逻辑。
- **底部横向区域**：汉字演化动画（五阶段变形）+ 毛笔笔画书写 + 语音控制。

## ⚡ 五、技术实现要点

- **3D 引擎**：Three.js (r128+)，为原子球添加 Rim Light 效果。
- **动画引擎**：GSAP (MorphSVG) 用于字形平滑过渡，ScrollTrigger 用于视差滚动。
- **汉字渲染**：Hanzi Writer 集成毛笔笔触效果。
- **语音系统**：Web Speech API 实现标准普通话发音。
